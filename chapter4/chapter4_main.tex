\section*{Выводы по главе~4}
\phantomsection\addcontentsline{toc}{section}{Выводы по главе~4}

При моделировании смазочного слоя подшипника скольжения
с~детерминированными микроструктурами возникают нелинейности
двух~принципиально различных типов. Первый~-- негладкая
нелинейность, порождаемая кавитацией в~массосохраняющей
постановке JFO: переключение между зонами полного заполнения
($p > p_{\mathrm{cav}}$, $\vartheta = 1$) и~кавитации
($p = p_{\mathrm{cav}}$, $\vartheta < 1$) формирует задачу
с~условиями комплементарности, граница которой заранее
неизвестна. Микроструктуры принципиально усложняют картину,
порождая множество фрагментированных кавитационных
<<островков>>, привязанных к~отдельным ячейкам микрорельефа.
Второй тип~-- гладкая нелинейность $\rho = \rho(p)$,
связанная со~сжимаемостью среды; она~устраняется итерациями
Пикара и~в~расчётах глав~5--7 не~используется, поскольку
смазка принята несжимаемой. Параметр~$\Lambda = L_1/L_2$
характеризует анизотропию уравнения Рейнольдса и~определяет
степень влияния локальных эффектов микрорельефа на~общую
картину давления. Постановка JFO обоснована как~базовая
для~всех~расчётов монографии.

Температура вводит ещё~одну гладкую нелинейность через
зависимость динамической вязкости от~температуры
$\mu = \mu(T)$. Убывающий характер этой зависимости~--
повышение температуры снижает вязкость и, как~следствие,
несущую способность~-- создаёт обратную связь между тепловым
и~гидродинамическим полями. Для~описания этого эффекта
сформулированы три~уровня учёта температуры: изотермическое
приближение ($T = T_0$, $\mu = \mathrm{const}$), приближение
эффективной температуры с~интегральным тепловым балансом
и~полная термогидродинамическая (THD) постановка, включающая
совместное решение уравнения Рейнольдса и~уравнения энергии.
Упрощённая конвективная модель с~граничным условием частичного
перемешивания на~входе представляет практически значимый
промежуточный вариант. В~расчётах глав~5--7 принято
изотермическое приближение, что~обосновано при~умеренных
скоростях скольжения и~нагрузках, когда температурный перепад
в~слое не~оказывает существенного влияния на~вязкость.

Случайная шероховатость поверхности принципиально отличается
от~детерминированных микроструктур: она~описывается не~явной
геометрией, а~статистическими параметрами~$R_a$, $R_q$,
$\lambda_c$. Определяющим критерием является число смазочного
слоя $\lambda_r = h_0/R_q$: при~$\lambda_r \gg 1$
шероховатость пренебрежимо мала,
при~$\lambda_r \approx 1$--$3$ она~существенно модифицирует
поле давления, при~$\lambda_r < 1$ рейнольдсовская модель
неприменима. Три~подхода к~учёту шероховатости~--
детерминированный (явное поле $\Delta h_{\mathrm{rough}}$),
стохастический (осреднённое уравнение с~коэффициентами
потоков~$\phi_p$, $\phi_s$ по~Патиру--Ченгу) и~пренебрежение
шероховатостью~-- различаются по~сложности реализации и~области
применимости. Для~типичных подшипниковых задач с~прецизионно
обработанными поверхностями глубина микроструктур существенно
превышает~$R_q$, что~обеспечивает $\lambda_r \gg 1$
и~обосновывает допущение о~гладких поверхностях, принятое
в~расчётах глав~5--7.

Динамика смазочного слоя рассмотрена в~контексте малых
колебаний вала относительно равновесного положения.
Линеаризация толщины зазора и~давления по~безразмерным
возмущениям~$\bar{\xi}$, $\bar{\eta}$ приводит к~системе
уравнений для~возмущённых полей давления, из~которых
интегрированием по~области смазочного слоя определяются
компоненты матриц жёсткости~$\mathbf{K}$
и~демпфирования~$\mathbf{C}$. Эти~матрицы в~общем случае
несимметричны, что~является принципиальным отличием
подшипников скольжения от~упругих опор и~источником
неконсервативных сил. Микроструктуры изменяют стационарное
и~возмущённые поля давления, а~следовательно,
все~компоненты~$K_{ij}$ и~$C_{ij}$~-- это~основной канал
влияния текстуры на~динамические характеристики. Устойчивость
роторно-подшипниковой системы определяется спектром матрицы
состояния~$\mathbf{A}$: система асимптотически устойчива тогда
и~только тогда, когда вещественные части всех~собственных
значений отрицательны.

Таким образом, глава~4 задаёт иерархию моделей-расширений
базовой изотермической JFO-постановки, сформулированной
в~главах~2--3. Каждое расширение~-- термовязкостное
(раздел~4.2), шероховатостное (раздел~4.3) и~динамическое
(раздел~4.4)~-- представляет собой самостоятельный модуль,
подключаемый к~базовому решателю по~мере необходимости.
В~главах~5--7 используется базовый уровень: изотермическое
приближение, несжимаемая смазка, гладкие поверхности,
стационарная постановка с~кавитацией JFO. Термовязкостное
и~шероховатостное расширения применяются при~наличии физических
предпосылок (высокие скорости, грубая обработка),
а~динамическое расширение с~вычислением коэффициентов
жёсткости и~демпфирования составляет предмет главы~8.
