\section{Дискретизация}

Дискретизации подлежат уравнение Рейнольдса (при полном заполнении зазора) и его обобщение JFO~\eqref{eq:jfo_conservation}. В обоих случаях используется консервативная (потоковая) форма записи: для каждой ячейки формулируется баланс потоков через её грани. Такой подход обеспечивает сохранение массы на дискретном уровне и естественным образом допускает обработку кавитационных зон.

Расчётная область покрывается прямоугольной равномерной сеткой с узлами~$(i,\,j)$ и шагами~$\Delta x_1$,~$\Delta x_2$ (введена в подразделе~2.3.3, формула~\eqref{eq:jfo_grid}). Потоки вычисляются на гранях ячеек: $i \pm 1/2$ по~$x_1$ и $j \pm 1/2$ по~$x_2$. Граничные условия на сетке реализуются в соответствии с разделом~3.1.

Консервативный баланс массы на ячейке~$(i,\,j)$ записывается в виде
\begin{equation}\label{eq:discrete_balance}
  \frac{(\vartheta h)_{i,j}^{n+1} - (\vartheta h)_{i,j}^{n}}{\Delta t}
  + \frac{F_{i+\frac{1}{2},j} - F_{i-\frac{1}{2},j}}{\Delta x_1}
  + \frac{G_{i,j+\frac{1}{2}} - G_{i,j-\frac{1}{2}}}{\Delta x_2}
  = 0,
\end{equation}
где $F = F_P + F_C$ -- суммарный поток через грань по~$x_1$ (напорный и переносной), $G = G_P + G_C$ -- аналогичный поток по~$x_2$. При полном заполнении ($\vartheta = 1$) уравнение~\eqref{eq:discrete_balance} сводится к классической дискретизации уравнения Рейнольдса. В стационарной постановке член по времени опускается.

\textit{Напорный (пуазейлевский) поток.}
Коэффициент проводимости определяется как
\[
  K = \frac{h^3}{12\mu}.
\]
Потоки на гранях аппроксимируются центральными разностями:
\begin{equation}\label{eq:flux_poiseuille}
  F_{P,\,i+\frac{1}{2},j} = -K_{i+\frac{1}{2},j}\,
    \frac{p_{i+1,j} - p_{i,j}}{\Delta x_1}, \qquad
  G_{P,\,i,j+\frac{1}{2}} = -K_{i,j+\frac{1}{2}}\,
    \frac{p_{i,j+1} - p_{i,j}}{\Delta x_2},
\end{equation}
Коэффициент~$K$ на грани вычисляется усреднением значений в соседних узлах (арифметическое или гармоническое усреднение); выбор конкретного способа не влияет существенно на результат при достаточном разрешении сетки.

\textit{Переносной (куэттовский) поток.}
Для уравнения JFO переносной поток записывается в виде
\begin{equation}\label{eq:flux_couette}
  F_{C,\,i+\frac{1}{2},j} = \frac{\vartheta_{\mathrm{up}}\,
    h_{i+\frac{1}{2},j}}{2}\,U_1, \qquad
  G_{C,\,i,j+\frac{1}{2}} = \frac{\vartheta_{\mathrm{up}}\,
    h_{i,j+\frac{1}{2}}}{2}\,U_2,
\end{equation}
где $\vartheta_{\mathrm{up}}$ берётся из «наветренного» узла по знаку соответствующей компоненты скорости~$U$ (upwind-аппроксимация). При полном заполнении ($\vartheta = 1$) множитель~$\vartheta_{\mathrm{up}}$ обращается в единицу. Толщина~$h$ на грани вычисляется арифметическим усреднением.

\textit{5-точечный шаблон для давления.}
Подстановка потоков в баланс~\eqref{eq:discrete_balance} при $\vartheta = 1$ (стационарный случай) приводит к линейной системе вида
\begin{equation}\label{eq:5point_stencil}
  a_P\,p_{i,j} = a_E\,p_{i+1,j} + a_W\,p_{i-1,j}
    + a_N\,p_{i,j+1} + a_S\,p_{i,j-1} + b_{i,j},
\end{equation}
где коэффициенты $a_E$,~$a_W$,~$a_N$,~$a_S$ определяются проводимостями~$K$ на соответствующих гранях; $a_P = a_E + a_W + a_N + a_S$; правая часть~$b_{i,j}$ содержит вклад куэттовского потока (и, при нестационарности, члена по времени). Система~\eqref{eq:5point_stencil} решается итерационно; выбор решателя описан в разделе~3.3.

Центральные разности для напорного потока обеспечивают второй порядок аппроксимации. Upwind-схема для переносного потока~$\vartheta$ имеет первый порядок, однако гарантирует устойчивость при доминировании конвективного переноса.
